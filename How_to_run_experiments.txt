HOW TO RUN EXPERIMENTS USING HYDRA
========================

PREREQUISITES
-------------
1. Ensure you are in the root directory

BASIC USAGE
-----------
# 1. Standard Run
# Runs dln/run.py using the default config specified in the @hydra.main decorator (see dln/run.py)
python -m dln.run

# 2. Run a specific config file
# Use --config-name (or -cn) to switch files. Do not include .yaml extension.
# Example: Runs using configs/single/test.yaml
python -m dln.run --config-name test

# 3. Run with a specific Experiment Name (for folder naming)
python -m dln.run name=my_custom_experiment

OVERRIDES (SINGLE RUN)
----------------------
# Syntax: key=value
# You can modify any value deep in the config tree using dot notation.

# Example: Change learning rate
python -m dln.run training.lr=0.005

# Example: Change multiple parameters
python -m dln.run training.lr=0.005 model.hidden_size=200

# Example: Change random seeds
python -m dln.run data.data_seed=42 training.model_seed=99

SWEEPS / MULTIRUN
-----------------
# Syntax: -m key=value1,value2,value3
# Use the -m (or --multirun) flag. Hydra will execute one run for every combination.

# Example: Sweep over 3 learning rates
python -m dln.run -m training.lr=0.1,0.01,0.001

# Example: Grid search (LR vs Hidden Size)
# This runs 3 x 2 = 6 total experiments.
python -m dln.run -m training.lr=0.1,0.01,0.001 model.hidden_size=50,100
